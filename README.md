## 📁 Cấu Trúc Thư Mục
root/

## 🔧 Cài Đặt & Chạy Dự Án

### 🖥️ Frontend

```bash
cd vite-project
npm install       # Cài đặt các gói phụ thuộc
npm run dev       # Khởi động server phát triển

Frontend mặc định chạy tại: http://localhost:5173

### 🖥️ Backend
```bash
cd backend
npm install       # Cài đặt các gói phụ thuộc
npm run start
